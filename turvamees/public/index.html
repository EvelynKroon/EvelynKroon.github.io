Turvamees Motion Guard

Приложение использует камеру, отслеживает направление движения объекта и воспроизводит отдельный звук для каждого движения.

Статус: ожидание запуска

Последнее движение: —

Звуки по направлениям

  • Влево: низкий короткий тон
  • Вправо: более высокий короткий тон
  • Вверх: высокий тон
  • Вниз: средний тон
  • Вперед: восходящий тон
  • Назад: прерывистый сигнал
