# Cyber Drive Adventure

Прототип 3D/2.5D игры про вождение с Node.js сохранениями.

## Что реализовано

- Камера от третьего лица позади машины.
- Управление: WASD / стрелки, пробел для нитро.
- Физика: ускорение, торможение, инерция при поворотах, столкновения.
- 5 последовательных уровней с постепенным усложнением.
- Препятствия, вражеские дроны, бонусы, роботы-мастера.
- HUD: уровень, очки, топливо, прочность, нитро, апгрейды.
- Меню: имя игрока, новая игра, продолжение, настройки графики.
- Сервер Node.js сохраняет прогресс по имени игрока.

## Запуск

1. Установить зависимости:

   npm install

2. Запустить сервер:

   npm start

3. Открыть в браузере:

   http://localhost:3000

## Формат сохранений

Сохранения лежат в data/progress.json и содержат:

- playerName
- currentLevel
- unlockedLevel
- score
- coins
- stats: health, fuel, nitro
- upgrades: acceleration, brakes, handling
- settings: graphics

## Ассеты

Папка public/assets содержит SVG-заглушки и asset-manifest.json.
Можно заменить их на PNG/SVG/GLB модели для генерации сцены в вашем пайплайне.
